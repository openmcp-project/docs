# Managed Control Planes

<div className="crd-header-container">
  <img src="/docs/img/cp2.png" alt="ManagedControlPlane" className="crd-header-icon" />
  <div className="crd-header-text">
    <p>Managed Control Planes are at the heart of OpenControlPlane. Simply put, they are lightweight Kubernetes clusters that store the desired state and current status of various resources. All resources follow the Kubernetes Resource Model (KRM), allowing infrastructure resources, deployments, etc., to be managed with common Kubernetes tools like kubectl, kustomize, Helm, Flux, ArgoCD, and so on.</p>

</div>
